# SmartElectrical Helper

SmartElectrical Helper is a Flutter-based mobile application that provides essential electrical engineering calculations.

## Features
- Ohm’s Law calculator (Voltage, Current, Resistance)
- Resistor color code decoder
- Power calculation (P = V × I)

## Technologies Used
- Flutter
- Dart
- Android Studio

## Description
This application implements real-time mathematical calculations with input validation and state management using Flutter’s Material components. 
It is designed as a simple and user-friendly engineering utility tool.
